## Phase Objectives
Develop tactile, interactive, and audio-visual tools to engage young learners (1-4 std).

## Working (Mechanism)
1. **Alphabet Tracing**: Uses HTML5 Canvas to track mouse/touch paths against letter templates.
2. **TTS**: Integrates Web Speech API for real-time text-to-audio narration.
3. **Show & Tell**: Utilizes a file upload component linked to a (mock) AI endpoint that identifies images and retrieves fun facts from RAG.
4. **Living Math**: Implements a drag-and-drop interface where visual objects represent numbers.

## Need (Rationale)
Younger students require sensory-rich interactions to maintain focus. Tracing helps with motor skills and letter recognition, while visual/audio feedback (TTS and Badges) reinforces learning through positive gamification.
