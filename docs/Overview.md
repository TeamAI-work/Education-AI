# Education-AI Overview

## Purpose
Education AI is a high-fidelity learning platform tailored for students from 1st to 12th standard. It provides specialized tools for each educational bracket, focusing on interactive learning, productivity, and career planning.

## Scope
- UI/UX Design of all 4 major sections (1-4, 5-8, 9-10, 11-12 std).
- Implementation of interactive tools: Alphabet Tracing, Living Math, Concept Mapping.
- Gamification via Streaks and Badges.
- Career path suggestions and exam strategy tools (Chapter Weightage).

## Stakeholders
- **Primary**: Students (1-12 std)
- **Secondary**: Educators and Parents
- **Developer**: Yash (GMS Project OS)

## Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router 7
- **Backend/Database**: Supabase (PostgreSQL, Auth, Storage)
